Breeze-OS-Test is a learning project exploring what it would take to build a simple operating system today compared to the early days of Windows and Linux. The goal was purely educational.

## Overview

- **Architecture:** x86_64
- **Languages:** C and Assembly
- **Toolchain:** GCC cross-compiler, GNU GRUB bootloader
- **Testing:** Oracle VirtualBox virtual machines

## Features Implemented

- Command-line input
- 64-bit paging memory access
- Interrupt handling
- Basic device drivers

## Status

This project was a learning experiment. The repository serves as a historical snapshot of early OS experimentation and learning in low-level programming.
